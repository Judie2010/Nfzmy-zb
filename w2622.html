核心主体 一、🔥 南方智媒云动态
南方报业传媒集团 · 广东省首个媒体行业大模型双备案平台
第二十二届深圳文博会闭幕：5天超280万人次参与，广东签约超100亿
南方日报5月26日
第二十二届中国（深圳）国际文化产业博览交易会于5月25日闭幕。全国6312家机构和企业参展，展出文化精品超12万件。广东展团现场签约和意向签约金额超100亿元。文博会吸引65个国家和地区、310家海外机构参展，约200家境内外媒体、近3300名记者参与报道，全网阅读量突破50亿人次。南方报业作为东道主核心参展媒体，以"四个一体化"系统性变革成果精彩亮相。
南方日报评论员："中国文化产业第一展"愈发出彩
南方+ / 南方日报5月26日
南方日报发表评论员文章，从三个维度总结文博会亮点：文化创新创造活力持续迸发，"首展""首秀""首发"不断，"+AI"和"AI+"科技元素突出；务实共赢的商业平台效应显著，广东展团签约超100亿元，9号馆文创圈单日意向成交额破亿元；融通中外的交流窗口作用强化，首设APEC经济体展区，文化出海"新三样"（网游、网剧、网文）表现亮眼。
南方观察｜逛文博会解锁"五感清远"
南方日报·清远观察 / 南方+5月29日
清远馆在文博会上以"五感清远"为体验主线，16家本土企业携非遗技艺和特色农产品参展，英德红茶区域公用品牌价值达57.32亿元。文博会期间举办农林文旅深圳专场招商推介会，签约8个重点项目及1组旅行社战略合作协议，签约总金额超37亿元，推动清远文旅产业融湾发展。
竞品追踪 二、⚔️ 竞品生态追踪

传播大脑（浙报集团）
当「龙虾」走进编辑部，媒体工作流如何被AI重构？
36氪（搜狐号）5月29日
传播大脑与腾讯云合作，基于ClawPro平台打造媒体智能助理"灵思超级助手"，采用"A-to-A"（Agent-to-Agent）模式，多智能体并行执行任务，实现强时效场景下30秒生成短视频。产品可个性化配置任务（新闻动态汇总、舆情追踪等），串联策、采、编、审、发全流程，强调"信源可查、过程可审、结果可控"的安全边界。文章指出AI能做到80分，但从80分到90分仍需人的判断。
深报智汇（深圳报业）
深圳报业改革成果文博会上受关注
深圳晚报 / 深圳新闻网5月27日
深圳报业集团在文博会上展示系统性变革成果：95%采编人员转向互联网生产，搭建"1总部+5中心"新架构，打造"1主报+1总端+1组号"党媒第一方阵。"问政深圳"工作室帖文回复率从70%提升到95%以上。"深报智汇"传媒大模型获中央网信办模型与算法双备案。与中国建设银行签署300亿元全方位战略合作，探索"文化传媒+产业平台+金融资本"模式。
文博会成全球文化产业"超级交易场"
读特新闻 / 深圳特区报5月27日
本届文博会以"交易力即生命力"为核心理念，5天展期吸引全球超13万名专业采购商到场，主分会场合同成交额同比增长超10%。首次设立下届展位预订处，老展商续订踊跃，人工智能展区意向交易总额突破亿元，强化了"中国文化产业第一展"的超级交易平台地位。
派生万物（上报/澎湃）
一键直出完整短剧，派生万物AI把创作门槛彻底踩平了！
澎湃新闻·湃客5月31日
派生万物推出AI短剧一站式"直出"平台，实现全流程自动生成（剧本理解→角色创建→镜头拆解→整集视频输出）。核心亮点：角色一致性保障（多场景形象固定+专属音色绑定）、接入可灵o3/Seedance 2.0等顶级视频模型、长篇剧本一键分集与资源复用。派生万物同时宣布专业级AI影视创作平台"星映"即将独立上线，从媒体工具向影视工业化基础设施延伸。
封面智媒云（华西/封面）
本周无重大更新。封面新闻10周年活动为4月28日，不在本报告周期内。其AI原生体系解决方案处于持续迭代中。
四川"一朵云"（天府融媒）
本周无重大更新。天府融媒联合体近期动态集中在4月的开放日活动，本周期内未发布新动态。
政策风向 三、🏛️ 行业政策与监管动态
广电总局印发《"微短剧精品创作传播计划"实施方案》
国家广播电视总局5月26日
国家广电总局办公厅印发方案，明确到2026年底基本建成微短剧精品创作传播扶持体系，全年推出千部优秀微短剧。实施"五个一批工程"（培根铸魂、反映时代新气象、地域文化特色、聚焦人间烟火、讲好中国故事五类精品），建立"总局引领、省局支持、平台激励"三位一体协同机制，同时配套部署有害低俗内容和侵权盗版专项治理，形成"扶持精品+整治乱象"组合拳。
中央网信办召开2026年全国信息化发展工作推进会
中国网信网5月28日
5月28日，中央网信办在深圳召开全国信息化发展工作推进会。中央宣传部副部长、中央网信办主任庄荣文出席并讲话。会议系统总结"十四五"信息化发展成就，部署"十五五"时期全国信息化发展工作。会议要求从战略规划、科技创新、产业生态、基础设施、重点应用、法规标准等维度体系化推进信息化发展，为媒体融合提供顶层政策支撑。
2026媒体融合发展论坛聚焦"系统变革 向新向优"
中国新闻网5月26日
由人民日报社主办的2026媒体融合发展论坛在武汉举行，主题"系统变革 向新向优"。中央和国家机关有关部委、中央和地方媒体、专家学者和企业代表等300余人参会。论坛设置"融合新动能：内容创新与生态重构"和"数智新图景：技术赋能与未来场景"两场平行分论坛，是"十五五"开局之年规格最高的媒体融合政策定调会议。
行业格局 四、📈 报业集团战略动态
拥抱人工智能 加快媒体数智化转型——AI与媒体变革分论坛深度记录
深圳晚报 / 深圳新闻网5月25日
5月22日，中宣部主办的文化强国建设高峰论坛"人工智能与媒体变革"分论坛在深圳举行。中宣部副部长莫高义强调主流媒体要拥抱AI、加快数智化转型，倡导价值向善并呼吁提供清洁语料。人民日报社杜尚泽强调"技术越强，人心越贵"，AI不能替代人的思考、立场和情怀。新华社副社长刘健提出"首先要懂，懂才能不犯错误；其次要用，用才能赶上时代"。
新华社总编室撰文：2026年全国两会报道的创新路径与经验启示
《传媒》2026年第7期 / 腾讯新闻5月26日
新华社总编室系统总结2026年全国两会报道创新经验：累计浏览量突破28亿次，微叙事透视宏大主题（如《最燃的接力》浏览量超2.6亿次），AI创意短视频《2026，山河向未来》采用"AI全流程全要素"创作模式总传播量超5000万次，以15个语种对外播发权威信息并被路透社、法新社等转引。为报业融媒的内容创新提供了央媒范本。
人民日报全国党媒平台智慧传播体系（党媒平台5.0）正式发布
全国党媒信息公共平台 / 云南网5月28日
2026媒体融合发展论坛期间，人民日报全国党媒信息公共平台正式发布"党媒平台5.0"。五大重点：智慧传播共同体（AI选题洞察—内容聚合—协同生产—精准分发全链条）、媒体智能体运营产品、党媒高质量数据集、区块链基础设施建设、行业服务生态。平台累计入驻单位达2042家，此前"党媒平台4.0"已入选中国报业创新发展案例。
技术前沿 五、🔬 融媒技术与产品前沿
AI赋能融媒创作 点亮智媒新生态
深圳特区报 / 搜狐5月26日
文博会现场展示AI技术深度融入媒体生产全链条的最新成果。深圳报业打造"深报智媒共创会客厅"，展示大疆、影石等智能采编设备，安克AI录音豆、深报AI平台等软硬件产品，覆盖从智能采编、一键创作出品到多语种智能译制、全流程智慧赋能的全业务场景，标志着媒体AI应用从概念展示进入全场景实景演练阶段。
第二十二届文博会释放AIGC新信号，十方融海解析个体生产力重构路径
凤凰网商业5月28日
文博会释放出AIGC与文化内容生产深度结合、个体生产力边界被重新定义的信号。十方融海CEO陈劢提出，AIGC的本质是生产关系的重构，OPC（一人公司）模式是AIGC时代的新型能力形态。十方融海已搭建"引擎—开源—智能体—终端"AI原生生态链路，为个体创作者提供能力支撑。对媒体行业意味着：内容生产力正从机构级配置向个体级赋能扩散。
传媒AIGC智能体落地指南：基于混元大模型的业务编排与分发重构
腾讯云开发者社区5月30日
腾讯云面向传媒行业推出AIGC智能体解决方案。落地路径分三阶：第一阶段内部员工赋能（部署预置智能体提效）；第二阶段自定义业务智能体（深度嵌套工作流）；第三阶段开放商业化（发布至腾讯元器平台，实现成本中心向利润中心转化）。方案依托腾讯混元生态，可一键将智能体分发至微信、企业微信、QQ等公域渠道，降低跨平台运维成本。
🎯 本周研判
📌 一句话总评：本周是中国报业融媒"系统性变革"加速落地的关键一周——武汉论坛定调"向新向优"、深圳AI分论坛凝聚"拥抱AI"共识、全国信息化推进会部署"十五五"顶层设计，三大会议密集共振，政策/技术/实践三线并进。
  1. 微短剧千部精品计划是内容侧的合规红利：广电总局"五个一批工程"+千部精品目标+侵权盗版专项治理，意味着微短剧赛道将从野蛮生长进入"持证上岗"阶段。主流媒体在内容审核、版权保护方面的天然优势将成为核心竞争力，建议提前布局微短剧精品内容管线。
  2. 智能体集群化是三阶段落地的最佳参照：腾讯云提出的"内部赋能→业务定制→开放变现"三阶路径，与传播大脑的"灵思超级助手"、派生万物的"星映"平台形成实践呼应。融媒技术团队可参考此框架，从预置智能体提效起步，逐步向业务深度嵌套和商业化演进。
  3. 党媒平台5.0的"智慧传播共同体"模式值得借鉴：人民日报提出的AI选题洞察—内容聚合—协同生产—精准分发全链条，本质是将零散的AI能力编排为完整的产品体系。南方智媒云作为省级平台，可思考如何在区域媒体协同中建立类似的智慧传播闭环。
